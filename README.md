# CC Object Presentation

Веб-приложение для генерации PDF-презентаций объектов недвижимости.

## Требования

- Node.js 18+
- pnpm (`npm install -g pnpm`)

## Установка и запуск

```bash
# 1. Установить зависимости
pnpm install

# 2. Запустить сервер и клиент одновременно
pnpm dev
```

После запуска:
- Фронтенд: http://localhost:5173
- Бэкенд: http://localhost:3001

## Структура проекта

```
cc-object-presentation/
├── shared/     — общие TypeScript типы
├── server/     — Node.js + Express + SQLite
│   └── data/   — БД и загруженные фото (создаётся автоматически)
└── client/     — React + TypeScript + Vite
```

## OpenAI API (для AI-реофлоу, опционально)

Создайте файл `server/.env`:
```
OPENAI_API_KEY=sk-...
```

## Как пользоваться

1. Открыть http://localhost:5173
2. Нажать «Новый объект» → заполнить данные → сохранить
3. Перейти на вкладку «Фото» → загрузить фотографии
4. Нажать «Создать слайды» → посмотреть предпросмотр
5. Нажать «Скачать PDF» → получить готовую презентацию
